# STOW TUI

## DESCRIPTION

_stow tui is a Terminal User Interface Program for [GNU STOW](https://www.gnu.org/software/stow/)_

## HOW TO INSTALL

Clone this repository,

```bash
git clone https://github.com/ypraw/stowtui.git
```

then run the installer script,

```bash
source ./install.sh
```
